**Module-level constants:**

```python
MAGIC = b'WGAV'                    # WireGuard Automate Vault
FORMAT_VERSION = 1
ARGON2_MEMORY_COST_KIB = 262144    # 256 MiB (NOT 256 -- KiB units!)
ARGON2_TIME_COST = 4
ARGON2_PARALLELISM = 4
ARGON2_HASH_LEN = 32               # 256-bit AES key
ARGON2_SALT_LEN = 16
GCM_NONCE_LEN = 12
DEFAULT_VAULT_DIR = Path.home() / ".wg-automate"
DEFAULT_VAULT_PATH = DEFAULT_VAULT_DIR / "vault.enc"
```

Add a prominent comment: `# CRITICAL: memory_cost is in KiB. 256 MiB = 262144 KiB. Passing 256 = catastrophically weak.`

**Inner vault JSON schema (schema_version: 1):**
```json
{
  "schema_version": 1,
  "server": {
    "private_key": "<base64>",
    "public_key": "<base64>",
    "port": 51820,
    "subnet": "10.0.0.0/24",
    "interface": "wg0",
    "dns_server": "1.1.1.1"
  },
  "clients": {},
  "ip_pool": {
    "subnet": "10.0.0.0/24",
    "server_ip": "10.0.0.1",
    "allocated": {}
  },
  "integrity": {}
}
```

**`_derive_key(passphrase: bytearray, salt: bytes) -> bytes`** (module-level function):
- Use `argon2.low_level.hash_secret_raw()` with `Type.ID`
- Pass `secret=bytes(passphrase)` (hash_secret_raw requires bytes, not bytearray)
- Use the module constants for memory_cost, time_cost, parallelism, hash_len
- Return the 32-byte derived key

**`_encrypt_vault(plaintext_dict: dict, passphrase: bytearray, hint: str | None = None) -> bytes`:**
- Generate salt with `os.urandom(ARGON2_SALT_LEN)`
- Generate nonce with `os.urandom(GCM_NONCE_LEN)` (SEC-06: fresh nonce per encrypt)
- Derive key via `_derive_key()`
- Build binary header using struct.pack:
  - 4 bytes: MAGIC (b'WGAV')
  - 1 byte: FORMAT_VERSION (0x01)
  - 4 bytes: ARGON2_MEMORY_COST_KIB (uint32 big-endian)
  - 4 bytes: ARGON2_TIME_COST (uint32 big-endian)
  - 4 bytes: ARGON2_PARALLELISM (uint32 big-endian)
  - 1 byte: salt length (16)
  - 16 bytes: salt
  - 1 byte: nonce length (12)
  - 12 bytes: nonce
  Total header: 47 bytes
- Serialize plaintext_dict to JSON with `json.dumps(d, separators=(',',':'))` (compact, no whitespace)
- Encrypt: `AESGCM(key).encrypt(nonce, plaintext_json_bytes, header)` -- header is AAD
- Wipe derived key bytes after use (create bytearray from key, wipe, del)
- Build final blob: header + struct.pack('>I', len(ciphertext)) + ciphertext
- If hint is not None, append hint encoded as UTF-8 after ciphertext
  Wait -- the hint goes in a SEPARATE FILE, not in the vault binary. Per research Pattern 7, hint is stored as `vault.hint` plaintext file beside `vault.enc`. Do NOT embed hint in the binary blob.
- Return the binary blob (header + ciphertext length + ciphertext)

**`_decrypt_vault(blob: bytes, passphrase: bytearray) -> dict`:**
- Validate minimum size (>= 51 bytes: 47 header + 4 ct_len)
- Validate magic bytes == MAGIC, else raise VaultTamperedError
- Parse header: version, memory_cost, time_cost, parallelism, salt_len, salt, nonce_len, nonce
- Extract ciphertext length (4 bytes uint32 big-endian at offset 47)
- Extract ciphertext (starts at offset 51, length from previous field)
- Derive key from passphrase + salt using parsed Argon2 params (use params from header, not module constants -- forward compatibility)
- Decrypt: `AESGCM(key).decrypt(nonce, ciphertext, header)` where header = blob[0:47]
- Catch `cryptography.exceptions.InvalidTag` and raise `VaultUnlockError("Vault unlock failed") from None` (locked decision: generic message, suppress traceback)
- Wipe derived key after use
- Return `json.loads(plaintext.decode('utf-8'))`

**`class VaultState`:**
- Holds the decrypted vault dict in memory
- `__init__(self, data: dict)`: store the dict, wrap all `*_key` and `psk` values in SecretBytes
- Properties for accessing server config, clients, ip_pool, integrity
- `to_dict(self) -> dict`: serialize back to dict (convert SecretBytes back to strings for JSON)
- `wipe(self)`: zero all SecretBytes in the state

**`class Vault`:**
- `__init__(self, vault_path: Path = DEFAULT_VAULT_PATH)`: store path
- Class method `create(cls, vault_path: Path, passphrase: SecretBytes, initial_state: dict, hint: str | None = None) -> "Vault"`:
  - **Passphrase length enforcement (VAULT-03):** At the very start of this method, before any file I/O:
    ```python
    raw = passphrase.expose_secret()
    if len(raw) < 12:
        raise ValueError("Passphrase must be at least 12 characters")
    ```
  - Create vault directory with 700 permissions (VAULT-02)
    - Unix: `os.chmod(dir, 0o700)`
    - Windows: use subprocess `icacls` to set SYSTEM+Administrators only (best-effort, log warning on failure)
  - Encrypt initial_state with passphrase
  - Atomic write (VAULT-05): use `tempfile.mkstemp(dir=parent)`, write blob, `os.fsync(fd)`, close fd, chmod 600 on tmp file (Unix), `os.replace(tmp, vault_path)`
  - If hint provided: write to `vault_path.with_suffix('.hint')`, print warning "WARNING: The passphrase hint is stored as plain text and is not protected."
  - Return Vault instance

- `open(self, passphrase: SecretBytes) -> VaultState` (VAULT-04):
  - Read vault file bytes
  - Call `_decrypt_vault(blob, passphrase.expose_secret())`
  - Return VaultState wrapping the decrypted dict

- Context manager `__enter__` / `__exit__`:
  - This is on VaultState, not Vault
  - `__exit__` calls `self.wipe()` in finally (VAULT-04)

- `save(self, state: VaultState, passphrase: SecretBytes)`:
  - Serialize state to dict via `state.to_dict()`
  - Encrypt with passphrase
  - Atomic write to vault_path with 600 permissions (VAULT-05)

- `change_passphrase(self, old_passphrase: SecretBytes, new_passphrase: SecretBytes)` (VAULT-07):
  - Decrypt with old passphrase
  - Re-encrypt with new passphrase (new salt, new nonce)
  - Atomic write
  - Wipe intermediate plaintext

- `verify_integrity(self, passphrase: SecretBytes) -> bool` (VAULT-08):
  - Attempt to decrypt vault (tests GCM tag + Argon2 salt integrity)
  - If successful, wipe decrypted state and return True
  - If InvalidTag, return False

- Static method `get_hint(vault_path: Path) -> str | None`:
  - Read `.hint` file if it exists, return stripped text
  - Return None if no hint file

**Atomic write helper `_atomic_write(path: Path, data: bytes, mode: int = 0o600)`:**
- Implement as module-level function (used by both create and save)
- `tempfile.mkstemp(dir=path.parent, prefix='.tmp_vault_')`
- Write data via `os.write(fd, data)`
- `os.fsync(fd)`
- `os.close(fd)`
- Unix: `os.chmod(tmp_path, mode)` BEFORE rename (file never world-readable)
- `os.replace(tmp_path, str(path))` (cross-platform atomic, works on Windows unlike os.rename)
- Unix: fsync parent directory
- Windows: skip parent fsync (NTFS journaling)
- Wrap in try/except BaseException: clean up tmp file on any failure

**Directory permissions `_ensure_vault_dir(dir_path: Path)`:**
- `dir_path.mkdir(parents=True, exist_ok=True)`
- Unix: `os.chmod(dir_path, 0o700)` (VAULT-02)
- Windows: best-effort `icacls` to restrict to SYSTEM+Administrators; wrap in try/except, log warning on failure (full Windows ACL support comes in Phase 2 permissions module)

**src/wg_automate/core/keygen.py:**

Implement `generate_keypair() -> tuple[SecretBytes, bytes]`:
- Returns `(private_key_b64, public_key_b64)` where private is SecretBytes and public is plain bytes
- Steps:
  1. `private_key = X25519PrivateKey.generate()` (KEYGEN-01: in-process, no subprocess)
  2. `raw_private = bytearray(private_key.private_bytes_raw())` (KEYGEN-02: bytearray immediately)
  3. `raw_public = private_key.public_key().public_bytes_raw()` (plain bytes, not secret)
  4. `private_b64 = base64.b64encode(bytes(raw_private))` -- standard base64, NOT url-safe
  5. `public_b64 = base64.b64encode(raw_public)`
  6. `wipe_bytes(raw_private)` (KEYGEN-04: wipe intermediate raw bytes)
  7. Wrap private_b64 in SecretBytes: `secret_private = SecretBytes(bytearray(private_b64))`
  8. Return `(secret_private, public_b64)`

Import: `from cryptography.hazmat.primitives.asymmetric.x25519 import X25519PrivateKey`
Import: `from wg_automate.security.secret_types import SecretBytes`
Import: `from wg_automate.security.secrets_wipe import wipe_bytes`

**src/wg_automate/core/ip_pool.py:**

Implement `class IPPool`:

`__init__(self, subnet: str)`:
- Parse with `ipaddress.ip_network(subnet, strict=False)` (friendly for user input like 10.0.0.1/24)
- Validate RFC 1918: `if not self.network.is_private: raise ValueError(f"Subnet {subnet} is not an RFC 1918 private range")`
- Store canonical form: `self.network = ipaddress.ip_network(subnet, strict=False)`
- Compute server IP: `self.server_ip = str(next(self.network.hosts()))` (always .1)
- Initialize empty allocation: `self._allocated: dict[str, str] = {}` (ip_string -> client_name)

`load_state(self, allocated: dict[str, str]) -> None`:
- Restore from vault state: `self._allocated = dict(allocated)` (copy to avoid aliasing)

`allocate(self, client_name: str) -> str` (IP-01, IP-02):
- Iterate `self.network.hosts()` starting from .1
- Skip `self.server_ip`
- Skip any IP already in `self._allocated`
- On first free IP: add to `self._allocated[ip_str] = client_name`, return ip_str
- If pool exhausted: `raise RuntimeError(f"IP pool exhausted for subnet {self.network}")`

`release(self, ip: str) -> None` (IP-03):
- `self._allocated.pop(ip, None)` -- immediate release, no grace period

`is_allocated(self, ip: str) -> bool`:
- `return ip in self._allocated`

`get_client_ip(self, client_name: str) -> str | None`:
- Reverse lookup: find IP allocated to this client name
- Return None if not found

`get_allocated(self) -> dict[str, str]`:
- Return `dict(self._allocated)` (copy for vault serialization)

Property `subnet_str -> str`:
- Return `str(self.network)`

--- add-client (CLIENT-01) ---
@cli.command("add-client")
@click.argument("name")
def add_client(name):
  1. Validate name: alphanumeric + hyphens only, max 32 chars. Use re.fullmatch
     r'^[a-zA-Z0-9-]{1,32}$'. Raise click.BadParameter if invalid.
  2. Collect passphrase via click.prompt("Vault passphrase", hide_input=True).
  3. Open vault via Vault context manager.
  4. Check name does not already exist in vault state. Abort if duplicate.
  5. Import generate_keypair from core.keygen; generate client keypair.
     Store private key (SecretBytes) in vault state under clients[name].
  6. Import generate_psk from core.psk; generate PSK. Store as SecretBytes
     in vault state under clients[name].
  7. Import IPPool from core.ip_pool; allocate next available IP. Store in
     vault state under clients[name].
  8. Import ConfigBuilder from core.config_builder; build client config
     (contains client private key, server public key, PSK, server endpoint,
     allowed IPs). Keep config string in memory only at this point.
  9. Import validator from security.validator; validate client config.
     Abort if validation fails.
  10. Write client config to ~/.wg-automate/clients/{name}.conf via
      atomic_write from security.atomic with mode=0o600.
  11. Compute SHA-256 of written file; store hash in vault state.
  12. Update server config to add new [Peer] block (via config_builder or
      direct patch). Re-validate. Write server config atomically.
  13. Reload WireGuard via: subprocess.run(["wg", "syncconf", "wg0",
      "<(wg-quick strip wg0)"], shell=True, check=True) — or equivalent
      per-platform command from the platform adapter.
  14. Log action="add-client", name=name, ip=allocated_ip (no key material).
  15. Display the QR code:
      from core.qr_generator import generate_qr_terminal, QR_DISPLAY_TIMEOUT
      print(generate_qr_terminal(client_config_str))
      print(f"QR will clear in {QR_DISPLAY_TIMEOUT} seconds...")
      time.sleep(QR_DISPLAY_TIMEOUT)
      click.clear()
  16. Wipe client_config_str from memory after display.

IMPORTANT: The Vault context manager must stay open for the entire operation
(steps 3-14). The vault is committed (context manager __exit__) only after
wg syncconf succeeds. If syncconf fails, raise the exception and let the
context manager discard changes.

--- remove-client (CLIENT-02) ---
@cli.command("remove-client")
@click.argument("name")
def remove_client(name):
  1. Collect passphrase via click.prompt("Vault passphrase", hide_input=True).
  2. Open vault via Vault context manager.
  3. Verify name exists in vault state clients dict. Abort if not found.
  4. Remove [Peer] block for name from server config. Re-validate server config.
     Write updated server config atomically.
  5. Reload WireGuard via wg syncconf (same as add-client step 13).
     This is the revocation moment — peer is disconnected immediately (CLIENT-02:
     no grace period).
  6. Release IP: update IPPool in vault state to mark IP as free.
  7. Purge from vault: delete clients[name] (private key, PSK, IP, config hash).
  8. Delete client config file ~/.wg-automate/clients/{name}.conf if it exists.
  9. Log action="remove-client", name=name (no key material).
  10. Print "Client '{name}' removed and revoked."
  </action>

@cli.command("list-clients")
def list_clients():
  1. Collect passphrase via click.prompt(hide_input=True).
  2. Open vault; read clients dict (names, IPs, config hashes).
  3. Run subprocess: ["wg", "show", "wg0", "dump"] with capture_output=True.
     Parse output to extract latest-handshake per peer public key.
  4. Print table: columns = Name, IP, Last Handshake.
     Map vault clients to wg show peers by matching public keys
     (public keys are non-secret; they appear in wg show output).
     NEVER print private keys, PSKs, or passphrase. CLIENT-03 invariant.
  5. If vault has clients but wg show shows no peers: print warning
     "WireGuard not running or no active peers."
  6. Log action="list-clients".

--- show-qr (CLIENT-04) ---
@cli.command("show-qr")
@click.argument("name")
@click.option("--save-qr", "save_path", default=None, type=click.Path(),
              help="Save QR to file (600 permissions)")
@click.option("--auto-delete", is_flag=True, default=False,
              help="Auto-delete saved QR file after 5 minutes")
def show_qr(name, save_path, auto_delete):
  1. Collect passphrase via click.prompt(hide_input=True).
  2. Open vault; retrieve client config for name. If not found: abort.
  3. Reconstruct client config string from vault state (private key, server
     public key, PSK, allocated IP, server endpoint). Use ConfigBuilder.
     Keep config string in memory only.
  4. Import generate_qr_terminal from core.qr_generator.
  5. Print QR to terminal.
  6. If save_path: call save_qr(config_str, Path(save_path), auto_delete).
  7. Start 60-second countdown: time.sleep(60); click.clear().
     Print "Terminal will clear in 60 seconds..." before sleeping.
  8. Wipe config_str from memory after clear.
  CLIENT-04: ASCII only, 60-second auto-clear.
  CLIENT-08: --save-qr writes with 600 perms; --auto-delete removes after 5 min.
